Medusa FAT read test file
=========================

This file is intentionally larger than one 512-byte sector so the stage 2 FAT code
can exercise multi-sector and multi-cluster reads instead of only reading a tiny
root-directory entry or a single short text file.

Line 001: The bootloader should be able to find this file in the FAT12 root directory.
Line 002: The filename is LARGE.TXT when stored in FAT 8.3 directory format.
Line 003: Reading this file should require following file data beyond the first sector.
Line 004: Each line adds predictable plain text for quick visual inspection.
Line 005: If output stops early, check sector advancement and cluster chaining.
Line 006: If output repeats, check file position and buffer refill logic.
Line 007: If output is scrambled, check segment:offset pointers passed to BIOS reads.
Line 008: If the file cannot be opened, check FAT name conversion and root scanning.
